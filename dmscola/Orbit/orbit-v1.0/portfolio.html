Design and Functionality in New Media and Electronic Spaces

This is a report on the study of design in electronic spaces and multimedia specifically on the internet. The report specifically introduces the concepts of UX design and UI design and details the relationship between functionality and design choices that a technical communicator must make. The project was completed for RPW 260.


Solar System CSS project

This is a recreation of the solar system only using CSS. This project was used as an introduction to cascading stylesheets and was created to test my abilities in manipulating objects on a webspace. The planets all rotate around the sun relative to their actual orbit times. Here, 10 seconds is equal to one orbit of Earth around the sun. Make sure to unzip and download/open each file in the 'ISD' folder or the page will not load correctly.


Dynamically Allocated Array and Linked List

These are two separate C++ programs written for my CS216 class. The first project is an example of dynamically allocated arrays using the 'new' operator. The program stores a user-specified number of intergers and then receives the inputs from the user's keyboard. The program then outputs the sum of the entire array.

The second program is an example of linked lists. The program creates a data structure with three fields. The user can navigate the program to insert a node at any location, delete a node at any location, search throughout the list for a specific node, or display the contents of the list.


Looping in C++

This is an informative report on looping while programming in C++. The report introduces the programming concept of looping to the reader, describes the three primary loops used in C++, and advises the user on when to use each loop in an program.

This report was designed with a target audience of programmers that are new to C++ and the programming construct of loops. It serves as an introductionary document and provides descriptions of the use of loops and images to show users examples of the code and the output it produces when compiled.
